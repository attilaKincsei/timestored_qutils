About

I got bored of the plain tables and started to make them a little prettier, I probably got a bit carried away

  • Snippets of code taken from the mailing list in particular functions created by Aaron Davies.
  • Any bugs / comments or ideas for additional functionality forward to mykdb@ryan-h.com

Improvements

  • Prettier tables with hover highlighting and client sortable columns
  • Hierarchical browsing menu that allows displaying more types
  • File Browser with Syntax highlighting of code
  • Time series graphs of data
  • Barcharts, column charts, treemaps, candlestick, pie charts available by providing google JSON formatted data
  • Screens showing system properties/connections etc.

It's also worth noting caching has been enabled for all static files
